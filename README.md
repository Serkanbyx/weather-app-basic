# Weather App (Basic)

A modern and user-friendly weather application. This basic version works with demo data.

## Features

- Modern and responsive design
- City search functionality
- Detailed weather information (temperature, wind, humidity, visibility)
- Quick access buttons
- Mobile-friendly interface
- Accessibility features

## Usage

1. Clone or download the project
2. Open `index.html` in a web browser
3. Enter a city name or select one of the quick access buttons

## Notes

This application uses demo data. To get real weather data, you will need to connect to the OpenWeatherMap API.

### Tips for Using OpenWeatherMap API

1. Get a free API key from [OpenWeatherMap](https://openweathermap.org/api)
2. Store your API key securely (use environment variables)
3. Add API calls in the `script.js` file
4. Never share your API key in a public repository
